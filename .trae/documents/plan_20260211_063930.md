由于项目目前缺少 `node_modules` 依赖包，您需要先安装依赖才能运行。

以下是完整的操作步骤和命令：

1.  **安装依赖 (Install Dependencies)**
    ```bash
    npm install
    ```

2.  **启动项目 (Start Project)**
    这条命令会同时启动前端 (Vite) 和后端 (Express) 服务。
    ```bash
    npm run dev
    ```

3.  **运行验证脚本 (Run Verification)**
    如果您想再次验证链接解析逻辑，可以运行：
    ```bash
    npx tsx scripts/verification_report.ts
    ```

**建议**：
您可以直接复制并在终端运行 `npm install`，安装完成后再运行 `npm run dev`。启动后，请访问终端提示的本地链接（通常是 `http://localhost:5173`）。
